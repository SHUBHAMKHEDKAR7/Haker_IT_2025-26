# Sainath Electricals & Hardware - Static Website

A modern, responsive static website for Sainath Electricals & Hardware, converted from React/TypeScript to pure HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Interactive Navigation**: Smooth section transitions and mobile-friendly menu
- **Product Catalog**: Dynamic product filtering by category
- **Contact Form**: Client-side validation and form handling
- **Customer Reviews**: Display of customer testimonials
- **Sign In Modal**: Authentication interface (frontend only)
- **SEO Friendly**: Semantic HTML structure

## Pages

1. **Home**: Hero section, featured products, why choose us, customer reviews
2. **Products**: Complete product catalog with filtering
3. **About**: Company story, mission, vision, and statistics
4. **Contact**: Contact form and business information

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # All CSS styles and responsive design
├── script.js           # JavaScript functionality and interactions
└── README.md           # This file
```

## Technologies Used

- **HTML5**: Semantic markup and accessibility
- **CSS3**: Modern styling with CSS Grid, Flexbox, and animations
- **Vanilla JavaScript**: No frameworks, pure JavaScript for all interactions
- **Responsive Design**: Mobile-first approach with media queries

## Getting Started

1. Simply open `index.html` in any modern web browser
2. No build process or dependencies required
3. All assets are self-contained

## Customization

### Products
Edit the `PRODUCTS_DATA` array in `script.js` to add, remove, or modify products.

### Reviews
Edit the `REVIEWS_DATA` array in `script.js` to manage customer reviews.

### Styling
Modify `styles.css` to change colors, fonts, spacing, or layout.

### Content
Update `index.html` to modify text content, contact information, or page structure.
